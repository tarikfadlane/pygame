# Zelda-Inspired Retro Audio Dashboard & Game

Bienvenue dans ce projet de jeu rétro développé avec **Pygame** ! Ce projet interactif combine une interface de contrôle audio (dashboard) avec un mini-jeu d'aventure inspiré de l'univers classique de Zelda.

## 🎮 Fonctionnalités

- **Mini-Jeu Interactif :** Déplacez votre personnage pour collecter des rubis et augmenter votre score.
- **Lecteur Audio Intégré :** Gérez la musique de fond directement depuis le jeu (Lecture, Pause, Rejouer).
- **Menu des Paramètres :** Une interface élégante en surimpression permettant de régler le volume sonore et d'activer/désactiver le mode muet.
- **Direction Artistique Rétro :** Utilisation d'une palette de couleurs soignée (Triforce Green, Rupee Green, Dark Forest Slate) rappelant l'ère 8-bit / 16-bit.

## 🛠️ Prérequis

Pour exécuter ce projet, vous devez avoir **Python** installé sur votre machine. Les dépendances externes se limitent à la bibliothèque `pygame`.

Vous pouvez l'installer via pip :
```bash
pip install pygame
```

## 🚀 Installation et Exécution

1. Clonez ce dépôt ou téléchargez les fichiers du projet.
2. Assurez-vous que les fichiers ressources nécessaires sont présents dans le répertoire racine :
   - `main.py` (Script principal)
   - `character.png` (Image du joueur)
   - `bg_music.mp3` (Musique d'ambiance)
3. Lancez le jeu via votre terminal :

```bash
python main.py
```

## 🕹️ Contrôles du Jeu

- **Déplacement :** `Touches Fléchées` ou `Z/Q/S/D` (W/A/S/D)
- **Musique :** `Espace` (Lecture/Pause), `R` (Recommencer)
- **Paramètres :** `Échap` (Ouvrir/Fermer le menu) ou Clic sur l'icône "Hamburger"
- **Volume :** Touches `+` et `-` de votre clavier
